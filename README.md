**입력받아 계산2**
----------------------------------------------------------
두 정수 a와 b를 입력 받은 후, a와 b를 곱한 값을 출력하는 프로그램을 작성해보세요.  
입력  
: 첫 줄에 두 정수 a, b가 공백으로 구분되어 주어짐  
출력  
: 첫 줄에 a와 b를 곱한 값을 출력합니다.

    #include <stdio.h>

    int main() {
    int a, b;
    scanf("%d %d", &a, &b);
    printf("%d", a*b);
    return 0;
    }
여기서 나는 scanf에서 **"&"** <<< 이거 빼먹음;;  

**입력받은 값 교체하기**
----------------------------------------------------------------------
정수 a, b를 입력 받고, a와 b의 위치를 교환한 후 출력하라  
입력  
: 첫 번째 줄에 정수 a, b가 공백을 사이에 두고 주어짐 (14 5)  
출력  
: 결과값 a와 b를 공백 사이에 두고 출력 (5 14)  

    #include <stdio.h>

    int main() {
    int a,b;
    int temp;
    scanf("%d %d", &a, &b);
    temp=a;
    a=b;
    b=temp;
    printf("%d %d", a, b);
    return 0;
    }
temp 위치 바뀌면 안됨.  
나는 temp썼음. 원래 교재의 정답은 아래 코드

    #include <stdio.h>

    int main() {
    // 변수 선언
    int a, b;

    // 입력
    scanf("%d %d", &a, &b);
    
    // 출력
    printf("%d %d\n", b, a);
    return 0;
    }
그냥 a,b 자리만 바꾼거임.

#2일차 **입력받아 계산3**
첫 번째 줄에 a와 b를 곱해준 값을 출력한다.
밑에는 내가 쓴 틀린 답^^

    #include <stdio.h>

    int main() {
    int a,b;
    printf("%d", a);
    scanf("%d", &a);
    printf("%d", b);
    scanf("%d", &b);
    printf("%d", a*b);
    return 0;
    }

해설^^

    #include <stdio.h>

    int main() {
    int a,b;
    scanf("%d %d", &a, &b);
    printf("%d\n", a*b);
    return 0;
    }
백준이나 코드트리 같은 채점 시스템에서는 입력값을 자동으로 넣어주기 때문에 "입력하세요" 같은 안내 문구가 필요 없고 오히려 출력에 방해가 될 수 있어서 생략하는 게 관례임!!
