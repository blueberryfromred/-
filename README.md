**입력받아 계산2**
두 정수 a와 b를 입력 받은 후, a와 b를 곱한 값을 출력하는 프로그램을 작성해보세요.  
입력  
>>>첫 줄에 두 정수 a, b가 공백으로 구분되어 주어짐
출력
>>>첫 줄에 a와 b를 곱한 값을 출력합니다.

    #include <stdio.h>

    int main() {
    int a, b;
    scanf("%d %d", &a, &b);
    printf("%d", a*b);
    return 0;
    }
여기서 나는 scanf에서 **"&"** <<< 이거 빼먹음;;
