**입력받아 계산2**
----------------------------------------------------------
두 정수 a와 b를 입력 받은 후, a와 b를 곱한 값을 출력하는 프로그램을 작성해보세요.  
입력  
>첫 줄에 두 정수 a, b가 공백으로 구분되어 주어짐
출력
>첫 줄에 a와 b를 곱한 값을 출력합니다.

    #include <stdio.h>

    int main() {
    int a, b;
    scanf("%d %d", &a, &b);
    printf("%d", a*b);
    return 0;
    }
여기서 나는 scanf에서 **"&"** <<< 이거 빼먹음;;  

**입력받은 값 교체하기**
----------------------------------------------------------------------
정수 a, b를 입력 받고, a와 b의 위치를 교환한 후 출력하라  
입력  
: 첫 번째 줄에 정수 a, b가 공백을 사이에 두고 주어짐 (14 5)  
출력  
: 결과값 a와 b를 공백 사이에 두고 출력 (5 14)  

    #include <stdio.h>

    int main() {
    int a,b;
    int temp;
    scanf("%d %d", &a, &b);
    temp=a;
    a=b;
    b=temp;
    printf("%d %d", a, b);
    return 0;
    }
temp 위치 바뀌면 안됨.  
나는 temp썼음. 원래 교재의 정답  

    #include <stdio.h>

    int main() {
    // 변수 선언
    int a, b;

    // 입력
    scanf("%d %d", &a, &b);
    
    // 출력
    printf("%d %d\n", b, a);
    return 0;
    }
